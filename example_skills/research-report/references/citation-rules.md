# 인용 규칙 & 검토 체크리스트 (허위 인용 방지)

## 인용 절대 규칙
1. 실제로 검색·확인한 자료만 인용한다. 존재하지 않는 논문/보고서/통계/URL을 만들지 않는다.
2. URL은 검색 결과에서 실제로 얻은 것만 쓴다. 그럴듯한 URL을 추측해서 적지 않는다.
3. 수치·통계는 출처의 표현을 그대로 옮기고, 발행일과 조사기관을 함께 남긴다.
4. 원문에서 확인 못 한 값은 `[확인 필요]`, 출처가 하나뿐인 주장은 `[단일출처]`로 표시한다.
5. 번역·요약 시 원문 의미를 왜곡하지 않는다.

## 사실 vs 해석
- **사실**: 자료에 직접 적힌 내용 → 출처와 함께.
- **해석**: 에이전트의 분석/추론 → "분석:" 또는 "해석:" 접두로 구분.

## 최종 검토 체크리스트
- [ ] 모든 핵심 주장에 출처가 연결됐는가
- [ ] 각 출처에 발행일(또는 확인일)이 있는가
- [ ] 핵심 주장이 2개 이상 출처로 교차검증됐는가 (안 되면 `[단일출처]`)
- [ ] 지어낸 URL/논문/통계가 없는가
- [ ] 사실과 해석이 구분돼 있는가
- [ ] 정량/정성 자료가 균형 있는가
- [ ] 편향·표본·기간·지역 한계가 명시됐는가
- [ ] 핵심 요약이 맨 앞에 있는가
