# 주식 뉴스 분석 — 규칙 & 자기점검 체크리스트

## 절대 규칙 (예외 없음)
1. 원문(기사)에 없는 수치·사실·발언·일정을 생성하지 않는다.
2. 모든 뉴스 항목에는 출처 URL과 출처명을 반드시 남긴다. 없으면 항목을 만들지 않는다.
3. 투자 매수/매도/보유 추천을 하지 않는다. ("사라", "팔아라", "지금이 매수 타이밍" 금지)
4. 주가 방향을 단정하지 않는다. ("반드시 오른다/떨어진다" 금지 → "긍정/부정/중립 영향 가능성"으로 표현)
5. 확인되지 않은 정보·루머는 넣지 않거나 `[확인 필요]`로 표시한다.
6. 기업의 미공개 중요정보·내부정보를 추정하거나 생성하지 않는다.
7. 사실과 해석(에이전트 판단)을 분리해 표기한다.

## 영향도 점수 기준 (1~5)
- 5: 실적/가이던스/대형 수주/규제 등 주가에 직접적·즉각적 영향이 큰 재료
- 4: 직접 관련 있으나 영향 시점/규모가 불확실
- 3: 산업 업황·경쟁사 동향 등 간접 영향
- 2: 관련은 있으나 영향 제한적
- 1: 참고 수준(맥락 정보)

## 최종 출력 전 체크리스트
- [ ] 모든 항목에 출처 URL이 있는가
- [ ] 발행일(또는 추정)이 표기됐는가
- [ ] 각 항목에 영향 방향 + 영향도 + 근거가 있는가
- [ ] 원문에 없는 숫자를 넣지 않았는가
- [ ] 투자 추천/단정 표현이 없는가
- [ ] 직접 영향과 간접(업황) 영향을 구분했는가
- [ ] 불확실 항목에 `[확인 필요]`를 붙였는가
- [ ] 맨 끝에 "본 분석은 투자 참고용이며 투자 자문이 아닙니다." 고지가 있는가
